# Email Signature Templates

Standard signatures for all agent emails.

## Standard Support Signature

```
Best,
PhotoVault Support
```

**When to use:** Most support responses

---

## Team Signature

```
Best,
PhotoVault Support Team
```

**When to use:** Escalations, team-involved responses

---

## Urgent/Priority Signature

```
We're on it!
PhotoVault Support Team
```

**When to use:** High-priority or time-sensitive issues

---

## Resolution Signature

```
Happy to help!
PhotoVault Support
```

**When to use:** When successfully resolving an issue

---

## Escalation Signature

```
We've got your back!
PhotoVault Support Team
```

**When to use:** When escalating to specialized team

---

## Follow-up Signature

```
Just checking in,
PhotoVault Support
```

**When to use:** Follow-up emails after initial response

---

## Variables

All signatures can include:
- `{{AGENT_NAME}}` - Specific agent name if needed
- `{{TICKET_ID}}` - Always include below signature

**Format:**
```
Best,
PhotoVault Support

---
Ticket ID: {{TICKET_ID}}
```
